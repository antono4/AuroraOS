Tentang Demo

Demo interaktif AuroraOS UI ini menampilkan berbagai fitur dan aplikasi yang tersedia di AuroraOS Custom ROM berbasis Android 17.

  • Lock Screen dengan jam real-time
  • Home Screen dengan app grid
  • Quick Settings panel
  • Berbagai aplikasi interaktif
  • Aurora wallpaper animasi

Interaksi

  • Geser ke atas Buka lock screen
  • Geser ke bawah Quick Settings
  • Tekan app Buka aplikasi
  • Tekan back Kembali ke home
  • M Toggle Quick Settings
  • ESC Tutup Quick Settings

Fitur UI

  • Aurora Theme
  • Glassmorphism
  • Smooth Animations
  • Real-time Clock
  • Dark Mode

🛠️ Instalasi

Build AuroraOS

Panduan singkat untuk membangun AuroraOS dari source

💻 Hardware Requirements

  • ✓ RAM: 16 GB minimum, 32 GB+ recommended
  • ✓ Storage: 500 GB minimum, 1 TB SSD recommended
  • ✓ CPU: Multi-core, 8+ cores recommended
  • ✓ Internet: Broadband 50+ Mbps

🔧 Software Requirements

  • ✓ OS: Ubuntu 20.04/22.04 LTS
  • ✓ Python: 3.8+
  • ✓ JDK: OpenJDK 17
  • ✓ Git: 2.0+

📝 Quick Setup

# Clone repository
git clone https://github.com/antono4/AuroraOS.git
cd AuroraOS

# Setup environment
chmod +x scripts/*.sh
./scripts/setup.sh

# Initialize & sync
repo init -u https://github.com/YOUR_USERNAME/manifest.git -b main
repo sync -c -j$(nproc)

# Build
source build/envsetup.sh
lunch aurora_oriole-userdebug
m -j$(nproc)

⚠️ Important

Build AuroraOS membutuhkan environment khusus dengan spesifikasi tinggi. Pastikan hardware Anda memenuhi persyaratan minimum sebelum memulai.

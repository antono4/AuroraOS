Privasi Terdepan

🌏

🔨 Langkah Build

Cara Build AuroraOS

Ikuti langkah-langkah berikut untuk membangun AuroraOS dari source code

1

Setup Environment

Install dependencies dan configure build tools

./scripts/setup.sh
2

Sync Source

Download AOSP + NusantaraProject repositories

repo sync
3

Customize

Kustomisasi branding, wallpaper, dan fitur

./scripts/customize.sh
4

Build ROM

Compile source code ke flashable ZIP

mka -j$(nproc)

Persyaratan Build

Pastikan sistem Anda memenuhi persyaratan minimum untuk build AuroraOS

Hardware

RAM 16 GB (32 GB+ recommended)

CPU Multi-core (8+ cores recommended)
Internet Broadband (Fiber recommended)

Software

OS Ubuntu 20.04/22.04 LTS
Python 3.8+
JDK OpenJDK 17
Git 2.0+
